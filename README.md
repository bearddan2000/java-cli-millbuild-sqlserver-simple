# java-cli-millbuild-sqlserver-simple

## Description
A java gradle build, that connects to sqlserver database.

## Tech stack
- java
- millbuild
  - 6.8.2

## Docker stack
- mcr.microsoft.com/mssql/server:2017-latest-ubuntu
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
